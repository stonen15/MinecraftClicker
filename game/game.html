How To Play


The goal of this game is to mine up diffent resources in order to gain more and more money. gather upgrades to make your mining more efficient and upgrade your tools. Keep mining and mining until you become so powerfull, you could slay a dragon.

Items

All of a set's items must be purchased before unlocking the next set of items.
Items are purchased from the shop and each item can only be purchasedonce.
Tools: When purchased, the tools will increase the value per click depending on the block it is used on.
After buying a tool, the upgraded version of the tool will be available for sale.
Each tool's upgrade will increase value per click by 1.
